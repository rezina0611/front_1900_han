DOM API란?
	문서의 객체 모델을 의미하여(Document Objedt Model)
	흔히 말하는 div, span, input과 같은 요소들을 모두 DOM이라고 한다

	HTML 태그들을 하나씩 객체화 한것이다
	HTML 페이지의 내용과 모양을 제어하기 위해 사용되는 객체이다
	HTML 태그 당 DOM 객체가 하나씩 생성된다
	HTML 태그의 포함관계에 따라서 부모 자식, 형제자매 관계로 구성된다


브라우저 객체 모델(BOM)
브라우저 객체 모델(Browser Object Model, BOM)은 문서 이외의 모든 것을 제어하기 위해 브라우저(호스트 환경)가 제공하는 추가 객체를 나타냅니다.
navigator 
location


DOM selector
   1) HttpCollection
      - document.getElementById() 
      - document.getElementsByClassName()
      - document.getElementsByTagName() 
   
   2) NodeList
      - document.getElementByName()
      - document.querySelector()
      - document.querySelectorAll()




이벤트

자주 사용되는 이벤트
   click - 요소를 클릭했을 때
   mouseover -요소에 마우스를 올렸을 때
   mouseleave -요소에서 마우스가 떠났을 때
   mousedown - 요소를 누르고 마우스를 아직 때기 이전 상태
   mouseup - 요소를 누르고 마우슬 땐 상태, 을 때 마우스 위치가 요소 위에 있어야함
   mousemove - 마우스가 움직일 때마다
   mousewheal - 마우스 휠이 이동되었을 때
   scroll - 스크롤의 변화가 있을 때
   focus - 활성화 되어있을 때
   blur - 포커스에서 벗어났을 때
   keypress - 키를 누른 순간
   keyup - 키를 누르고 있다가 땐 순간
   keydown - 키를 누르고 때기 이전인 순간
   change - 폼 필드의 속성이 변경되었을 때
